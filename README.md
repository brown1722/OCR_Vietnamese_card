# OCR Vietnamese Card
A FastAPI-based Optical Character Recognition (OCR) system for extracting key information from Vietnamese Citizen ID cards (CCCD/CMND). The model combines YOLOv11 for text region detection and VietOCR for text recognition, optimized for Vietnamese language and document layout.

# Setting 

```
git clone https://github.com/brown1722/OCR_Vietnamese_card.git

cd OCR_CCCD

pip install -r requirements.txt
```
